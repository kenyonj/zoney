# Zoney

This gem allows you to communicate with the [MonoPrice 6 Zone
Amplifier](https://www.monoprice.com/product?p_id=10761) via a serial port
connection

## Development

After checking out the repo, run `bin/setup` to install dependencies. Then, run `rake spec` to run thekjhgnbbkhe version, push git commits and tags, and push the `.gem` file to [rubygems.org](https://rubygems.org).
